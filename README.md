# pymsodbc

A ready-to-use Docker image with Python3 and Microsoft ODBC Driver for SQL Server.

## Run

```shell
docker run -it vidolin/pymsodbc:latest
```

## Driver path

/opt/microsoft/msodbcsql17/lib64/libmsodbcsql-17.5.so.1.1
